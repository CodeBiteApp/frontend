# Walkthrough: 상점 알림 및 내 보관함 UI 추가

**작성일**: 2026-05-16
**관련 컴포넌트**: `app/(tabs)/reward.tsx`, `store/useItemStore.ts`

## 구현 개요
상점 화면 내에서 내가 보유한 아이템을 확인하고 장착/해제할 수 있는 **내 보관함** 기능과, 아이템 구매 성공 시 즉각적인 피드백을 주는 **구매 알림** 기능을 추가했습니다.

## 상세 변경 내용

### 1. 상점 / 내 보관함 탭 전환 기능 (`reward.tsx`)
- 화면 최상단 헤더(도비의 상점) 아래에 `[상점] | [내 보관함]`을 전환할 수 있는 Segmented Control 형태의 버튼을 추가했습니다.
- `useState<"SHOP" | "INVENTORY">`를 통해 상태를 관리합니다.
- `useFocusEffect` 내에서 `fetchShopItems()`와 더불어 `fetchInventory()`를 함께 호출하여 항상 최신 보관함 데이터를 유지하도록 개선했습니다.

### 2. 내 보관함 (Inventory) 뷰 렌더링
- **스트릭 보호권**: `protectorCount` 상태를 사용하여 현재 보유 중인 소모품(보호권) 개수를 단일 카드로 표시합니다.
- **꾸미기 아이템 (배너, 아이콘 등)**: 
  - `useItemStore`의 `inventory`(보유 아이템 리스트)를 순회하며 카드를 렌더링합니다.
  - 각 아이템 카드 하단에 **[장착하기] / [해제하기]** 버튼을 추가했습니다.
  - 버튼을 누르면 `toggleEquip(itemId, !isEquipped)`가 호출되어 즉시 상태가 반영됩니다.
  - 장착 중인 아이템은 노란색 테두리(`cardEquipped` 스타일)로 시각적으로 강조됩니다.

### 3. 구매 완료 알림창 추가
- 아이템 구매 로직인 `handleBuy` 함수 내에 구매 성공 분기 처리를 보강했습니다.
- `buyItem()`이 성공적으로 `true`를 반환하면 구매 모달을 닫고, `Alert.alert("구매 성공", "구매완료 되었습니다.")`를 띄워 사용자에게 확실한 피드백을 제공합니다.

## 결과 및 효과
- 기존에 상점(구매 기능)만 있던 화면이 탭 형태로 확장되어, 새로운 화면(스크린) 이동 없이도 내 아이템을 관리할 수 있게 되어 UX가 크게 향상되었습니다.
- 사용자는 본인이 구매한 꾸미기 아이템을 손쉽게 장착/해제하며 시각적인 즐거움을 얻을 수 있습니다.
